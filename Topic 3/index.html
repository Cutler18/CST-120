CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of the web development languages including HTML, CSS, and JavaScript. The course focuses on foundation required to build complex dynamic web applications. Prerequisite: none.

Topics and Objectives

Week 1

HTML

In this topic, students are introduced to the HTML markup language. They will learn the basic structure, the basic elements of an HTML document, and how to work with text and basic controls. The topic ends with learning about the developer tools available, as well as how web servers work to help support development of HTML based applications.

  1. Develop webpages with basic elements and structure that include, but are not limited to, document, head, and body tags.
  2. Develop webpages with text that include, but are not limited to, headings, paragraphs, formatting, fonts, and color tags.
  3. Develop webpages with components that include, but are not limited to, links, images, and lists tags.
  4. Investigate how a web server functions and identify differences between localhost, file URL, and domain names in the URL.
  5. Investigate the developer tools available as a webpage developer.

Readings

  • none

Week 2

Advanced HTML

In this topic, students continue to expand their knowledge of the HTML markup language, learning some of the more advanced features as they discover how to work with layouts, graphics, and media content. The topic concludes with learning how to make HTML pages that are ADA compliant.

  1. Develop webpages with layouts that include, but are not limited to, Blocks/Div>Span, tables, and layout tags
  2. Develop webpages with graphics that include, but are not limited to, Image, Canvas, and SVG tags.
  3. Develop webpages with media that include, but are not limited to, video and audio tags.
  4. Explain how and why ADA compliance is achieved with ADA guidelines.

Readings

  • none

Week 3

CSS

In this topic, students learn Cascading Style Sheets and how to apply styling to HTML pages using fonts, colors, and formatting. The topic finishes with learning how to apply backgrounds, transitions, and animations to HTML pages.

  1. Develop webpages with CSS Basics Styles that include, but are not limited to, Inline, Internal, External, and Selectors styles.
  2. Develop webpages with text that include, but are not limited to, built-in fonts, custom fonts from external sources, color, and alignment styles.
  3. Develop webpages with formatting and layouts that include, but are not limited to, backgrounds, borders, margins, padding, width/height, and the Box Model styles.
  4. Develop webpages that include, but are not limited to, transitions and animations styles.
  5. Apply modern styling and formatting techniques to enhance the presentation of webpages.

Readings

  • none

Week 4

JavaScript

In this topic, students are introduced to the JavaScript programming language. This programming language is used as the foundation for client-side programming on webpages. Students learn the basic language syntax, how to work with variables, make decisions, perform repetitive operations, and create reusable functions.

  1. Develop code that integrates basic JavaScript using Inline statements in a webpage.
  2. Develop code that leverages variables and data types in a webpage.
  3. Develop code that leverages operators (math, assignment, comparison, logical, boolean, and string) in a webpage.
  4. Develop code that leverages arrays in a webpage.
  5. Develop code that uses reusable functions in a webpage.

Readings

  • none

Week 5

Advanced JavaScript

In this topic, students continue to expand their knowledge of the JavaScript programming language, learning how to create objects, work with data as arrays, and handle events from DOM objects from their HTML pages. The topic finishes with an introduction to the JavaScript Object Notation (JSON), emphasizing the advanced features of newer updates to the JavaScript programming language, and how this programming language is used alongside current web application frameworks that are used as the foundation to build modern, dynamic web applications.

  1. Develop code that integrates basic JavaScript from external files in a webpage.
  2. Develop code that uses decision and loop constructs in a webpage.
  3. Develop code that leverages JavaScript objects in a webpage.
  4. Develop code that handles events in a webpage.
  5. Explain what JSON is and how this data format is used to build web applications.
  6. Investigate advanced features found in recent updates to the JavaScript programming language.
  7. Investigate recent JavaScript Web Application Frameworks that are available to build dynamic web applications.
  • none

Week 6

Web Development Libraries

In this topic, students learn to use web development libraries to add dynamic, rich, and compelling content and user interfaces to their webpages. Students will also be introduced to the concept of responsive webpages, what this means, and how they can support this concept when building webpages.

  1. Develop webpages that support a responsive design.
  2. Develop webpages that leverage the use of jQuery, including, but not limited to, the use of Selectors, Events, DOM Manipulation, AJAX, and Data Table features.
  3. Develop webpages that leverage the use of Bootstrap, including, but not limited to, the use of the Grid System and various UI Components (Buttons, Text, Alerts, Tables, Forms, and NavBar) features.
  • none

Week 7

Advanced Web Development

In this topic, students learn that static webpages have their limits and that they often need to be deployed on web servers sometimes backed with a server-side programming language or database. This topic, at minimum, will introduce students to the HTTP protocol, the client server model, HTML forms, and how to implement forms processing to expand on the capabilities of the webpages they have built in previous topics. Lastly, students are introduced to the basic concepts required for future web application development courses where formal dynamic web application frameworks are used.

  1. Investigate HTTP protocol and the client server model.
  2. Develop webpages with HTML forms.
  3. Develop webpages that integrate forms processing on the server.
  • none
